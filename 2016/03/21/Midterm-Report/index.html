Introduction

The dataset we are using is from Yelp Dataset Challenge. There are six subsets in total: business, review, user, check-in, tip and photos. And all the data are in json format. We have been using the business, review and user datasets, which contains 77,445 businesses, 2,225,213 reviews and 552,339 user profiles. Visualization

Our final project uses datasets from Yelp and wants to create a sentiment list specifically for Yelp reviews (something similar to AFINN-111, but fits Yelp better). Visualization above is a word cloud which shows the top 163 words (positive words) which show up in 5-star (most positive) reviews the most frequently. The bigger the word is, the more frequently it shows up in 5-star review, and it tends to be more positive.
We perform the steps below to get rid of neutral words which shows up a lot but don’t have a positive or negative meaning with it, such as “food”, “service”:

  1. Lowercase, delete all the stopwords and punctuation as well as all reviews in other language.
  2. Take all the reviews scores 5 and 1, find the top 500 words that are most frequently shown in the reviews, named as wordlist1 (top 500 words in 1-star review)and wordlist5 (top 500 words in 5-star review)
  3. Compare the two lists, delete any words that exist in both list
  4. The words left in the wordlist5 are what we want.
